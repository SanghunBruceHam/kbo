📊 매트릭스 해석 가이드

🎨 셀 색상 가이드
  • ✓ 초록: 순위 확정 (이미 달성)
  • 연초록: 매직넘버 (승수로 표시)
  • 노랑: 경고 (트래직넘버 근접)
  • 연빨강: 트래직넘버 (패수로 표시)
  • ✗ 빨강: 순위 달성 불가능
  • 갈색: 자력으로 확정 불가
📊 매트릭스 구조
  • 세로축: 각 팀별 현재 상태
  • 가로축: 최종 순위 (1위~10위)
  • 포스트시즌: 1~5위 (금색 경계선)
  • 숫자 의미: 해당 순위 달성에 필요한 승수
✅ 표시 방식
  • 숫자: 매직넘버 (해당 순위 확보 필요 승수)
  • ✓ "확보": 해당 순위 이미 확보
  • ✗ "X위 불가": 해당 순위 달성 불가능
  • 5위 옆 주황 구분선: 포스트시즌 진출 기준선
* 이 매트릭스는 나무위키를 참고하여 만들었습니다

📊 기본 지표

  • 게임차: 1위와 승수 차이

  • 최대승수: 전승시 최종 승수
🏆 1위 탈환 계산법
  • 1위팀 조건: 추격팀 전승시 1위가 져야할 경기수
  • 트래직넘버: 1위팀이 내 최대가능 승률을 넘기 위해 필요한 승수
  • 계산방식: 승률 비교 기준으로 1위 추월 가능성 분석
  • "불가능": 전승해도 1위팀 최저가능 승률을 넘을 수 없음
  • "역대기준": 87승 달성 필요 승률 및 가능 여부 (와일드카드 도입 후 평균)
📈 승률 난이도: ■ 0-45% ■ 45-55% ■ 55-70% ■ 70%+
⚾ 타이브레이커 표시: TB = 타이브레이커 필요 + = 상대전적 우위 - = 상대전적 열세 ⚾ = 타이브레이커 가능

  • 잔여경기: 남은 경기 수
  • 최대승수: 전승시 최종 승수
🏟️ 포스트시즌 진출 조건 계산법
  • PS 매직넘버: 5위팀(없으면 4위팀) 최대가능 승률을 넘어 PS 진출 확정하기 위해 필요한 승수
  • 계산방식: 승률 비교 기준으로 포스트시즌 진출 확정점 분석
  • "자력 불가": 잔여경기 전승해도 5위팀 승률을 넘을 수 없음 (괄호 안 실제 필요 승수)
  • PS 트래직넘버: 나의 최대가능승수 - (5위팀 현재승수 - 1)
  • "탈락": 트래직넘버가 0 이하로 포스트시즌 진출 불가능

  • 역대 기준 트래직넘버: 70승 달성 불가능까지 남은 패수
  • 필요승률: 잔여경기 목표 승률
